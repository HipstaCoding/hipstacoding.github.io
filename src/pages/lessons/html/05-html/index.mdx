## Дочерние селекторы

Комбинация из двух и более селекторов идущих через пробел:

```html
<ul class="section-list">
  <li>section 1</a>
  <li>section 2</a>
</ul>
```

```css
.section-list li {
  text-transform: uppercase;
}
```

</section>

### Не больше двух-трех селекторов!

#### Внегласное правило CSS 5.2
### Предпочтение отдается отдельному классу

</section>

### Когда использовать?

- Когда лень давать класс элементу
- Могут быть дочерние теги только одного типа (например: `ul li`)
- Один и тот же компонент отображается в зависимости от родителя

### Псевдокласс :nth-child

<section>

#### Режим 1 - число

```css

  color: yellow;
```

#### Режим 2 - четный/неченый
##### even/odd

```css

  color: yellow;

  color: blue;
}
```

## Псевдоэлементы

</section>
<section>

Псевдоэлемент - селектор CSS, который стилизует то, что не объявленно напрямую в HTML. 

```css
ul li::before {
  content: "-";
}

.width-image::after {
  content: "";
  display: block;
  background-image: url('path/to/image.png');
  width: 100px;
  height: 100px;
}
```

</section>
<section>

### Внегласное правило CSS 5.3

Псевдоэлемент пишется через два двоеточие `::`, но работает можно писать и как псевдокласс `:after`

</section>
